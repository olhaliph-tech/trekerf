## 🚀 ТОП 10 ФУНКЦІЙ

### 1️⃣ 📝 **Приватні Нотатки**
Додавайте особисті думки та записи до кожної книги.
```javascript
LibraryExtended.addBookNote(bookId, "Текст", pageNum)
```

### 2️⃣ 🔖 **Закладки та Цитати**
Зберігайте улюблені цитати з вказанням сторінок.
```javascript
LibraryExtended.addQuote(bookId, "Цитата", pageNum)
```

### 3️⃣ 🏷️ **Кольорові Теги**
Організуйте книги за тегами з красивими кольорами.
```javascript
LibraryExtended.addColorTag(bookId, "класика", "#E8B4B8")
```

### 4️⃣ ⭐ **Швидкі Теги**
Однокліком позначте: "обов'язково прочитати", "рекомендована", тощо.
```javascript
LibraryExtended.addQuickTag(bookId, 'обов_прочити')
```

### 5️⃣ 🎁 **Статус Подарунків**
Записуйте, хто та коли подарував вам книгу.
```javascript
LibraryExtended.setAsGift(bookId, "Від кого", "Причина")
```

### 6️⃣ 📊 **Аналітика Книги**
Дивіться детальну статистику: прогрес, темп, точна дата закінчення.
```javascript
LibraryExtended.getBookAnalytics(bookId)
// { progressPercent, pagesPerDay, estimatedFinishDate, ... }
```

### 7️⃣ 🎨 **Кастомізація Карточок**
Налаштуйте що показувати на карточках книг.
```javascript
LibraryExtended.setCardSize('large')
LibraryExtended.setCardDisplayOptions({ title: true, author: true, ... })
```

### 8️⃣ 🔔 **Смарт-Нагадування**
Автоматичні алерти для книг, яких потребують уваги.
```javascript
LibraryExtended.getSmartReminders()
// [{ type, bookId, message, priority }, ...]
```

### 9️⃣ 📚 **Колекції Книг**
Організуйте книги у власні колекції.
```javascript
const coll = LibraryExtended.createCollection("Улюблені автори")
LibraryExtended.addBookToCollection(collId, bookId)
```

### 🔟 👥 **Порівняння з Друзями**
Дивіться як ви читаєте в порівнянні з друзями.
```javascript
LibraryExtended.getBookComparisonStats(bookId)
// { yourProgress, averageProgress, yoursAhead, ... }
```

Натисніть **F12** у браузері та перейдіть на таб **Console**

### Крок 2: Скопіюйте Код
Виберіть будь-який приклад з **LIBRARY_EXTENDED_EXAMPLES.md**

### Крок 3: Вставте та Запустіть
Вставте код в консоль та натисніть Enter


## 💾 ЗБЕРЕЖЕННЯ ДАНИХ

Всі дові автоматично зберігаються у **localStorage** при кожному оновленні.

Структура для кожної книги:
```javascript
{
  notes: [],        // Нотатки
  quotes: [],       // Цитати
  bookmarks: [],    // Закладки
  colorTags: [],    // Теги з кольорами
  quickTags: [],    // Швидкі теги
  gift: {},         // Інформація про подарунок
}
```
