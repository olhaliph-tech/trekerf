## 🧪 Тестування

### Ручна перевірка:

✅ **Додавання заметки**
```
1. Відкрийте будь-яку книгу
2. Перейдіть на вкладку "Заметки"
3. Напишіть "тест"
4. Натиснувши "Додати"
5. Має з'явитися в списку ✓
```

✅ **Оцінка при статусі "Прочитано"**
```
1. Змініть статус на "Прочитано"
2. Перейдіть на вкладку "Оцінка"
3. Натиснувши на 5 зірок
4. Повинно зберегтися ✓
```

✅ **Оцінка при іншому статусі**
```
1. Статус = "Бажаю прочитати"
2. Спробуйте ставити оцінку
3. Повинна помилка: "⚠️ Можна ставити..." ✓
```

✅ **Отзыв зберігається**
```
1. Статус "Прочитано"
2. Напишіть в полі від­гука
3. Натиснувши "Збе­регти"
4. Сповіщення "✅ Отзыв сохранен!" ✓
```

✅ **Перезагрузка сторінки**
```
1. Додайте все (заметку, оцінку, відгук)
2. Перезагрузіть сторінку (F5)
3. Все повинно залишитися ✓
```

## 📞 Помічь

**Якщо щось не працює:**

1. **Заметка не зберігається?**
   - Перевірте, текстове поле не пусте
   - Натиснить "➕ Додати заметку"

2. **Не можу ставити оцінку?**
   - Перевірте статус = "Прочитано"
   - Якщо ні - змініть на вкладці "Інформація"

3. **Від­гук не зберігається?**
   - Натиснить "💾 Зберегти від­гук"
   - Дочекайніся "✅ Сповіщення"

4. **Усе видалилось?**
   - Можливо очищено localStorage?
   - DevTools → Application → Local Storage → перевірити
