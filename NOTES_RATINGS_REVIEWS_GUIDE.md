# 📝 Заметки, Оцінки та Відгуки - Користувацький Посібник

## 🎯 Новые функции

В версии 2.1 добавлены три мощные функции для управления книгами:

### 1. **📝 Заметки к книге**
Дозволяє користувачам залишати особисті заметки під час читання книги.

#### Як користуватися:
1. Відкрийте книгу (натиснувши "Деталі")
2. Перейдіть на вкладку **📝 Заметки**
3. Введіть вашу заметку в текстове поле
4. Натиснувши **➕ Додати заметку**

#### Функциональність:
- ✅ Додавання невідомої кількості заметок
- ✏️ Редагування заметок (натиснути "✏️ Редагувати")
- 🗑️ Видалення заметок (натиснути "🗑️ Видалити")
- 📅 Кожна заметка має дату створення
- 💾 Все автоматично зберігається в localStorage

#### Приклад використання:
```
Заметка: "Цікавий фрагмент про головного героя на сторінці 42"
Дата: 05.04.2026

Заметка: "Нез'ясована сюжетна лінія - потребує уточнення в кінці"
Дата: 06.04.2026
```

---

### 2. **⭐ Оцінка книги (Rating System)**
Дозволяє оцінити прочитану книгу від 1 до 5 зірок.

#### Як користуватися:
1. Прочитайте книгу (статус повинен бути "Прочитано")
2. Відкрийте книгу (натиснувши "Деталі")
3. Перейдіть на вкладку **⭐ Оцінка**
4. Натиснувше на потрібну кількість зірок (1️⃣ до 5️⃣)

#### Особливості:
- ⭐⭐⭐⭐⭐ - Відмінна книга
- ⭐⭐⭐⭐ - Дуже добра
- ⭐⭐⭐ - Добра
- ⭐⭐ - Задовільна
- ⭐ - Слаба

#### Обмеження:
- ⚠️ Можна ставити оцінку **тільки** коли книга має статус "Прочитано"
- Якщо змінити статус на інший, оцінка залишиться, але нова оцінка не може бути поставлена

---

### 3. **💬 Відгук про книгу (Review)**
Напишіть детальний текстовий відгук про прочитану книгу.

#### Як користуватися:
1. Прочитайте книгу (статус "Прочитано")
2. Відкрийте книгу
3. Перейдіть на вкладку **⭐ Оцінка**
4. В секції **Ваш відгук** залиште текст
5. Натиснувши **💾 Зберегти відгук**

#### Що писати у відгуку:
- 📖 Ваші враження про сюжет
- 👤 Думка про персонажей
- 💭 Що вас най більше вразило
- 🎯 Рекомендуєте ли книгу іншим
- ⚖️ Порівняння з іншими книгами автора

#### Приклад відгуку:
```
Чудова книга! Мені дуже сподобався динамічний сюжет та глибокі персонажі.
Автор чудово розвиває историчний контекст. Рекомендую всім любителям фентезі.
Прочитав за выходних, не міг відірватися від сторінок!
```

## 📊 Де зберегаються дані?

Все дані автоматично зберігаються в **localStorage** вашого браузера:

| Тип даних | Де зберегаються | Постійність |
|-----------|-----------------|-------------|
| Заметки | `book.notes[]` | Постійно |
| Оцінка | `book.rating` | Постійно |
| Відгук | `book.review` | Постійно |
| Дата відгука | `book.reviewDate` | Постійно |

Все автоматично синхронізується при кожній дії!

## 🔧 API для розробників

Якщо хочете розширити функціональність, доступні ці функції:

```javascript
// Заметки
addNote(bookId, noteText)          // Додати заметку
removeNote(bookId, noteId)         // Видалити заметку
editNote(bookId, noteId, newText)  // Редагувати заметку
getNotesByBook(bookId)             // Отримати всі заметки

// Оцінки
setBookRating(bookId, rating)      // Встановити оцінку (1-5)
getBookRating(bookId)              // Отримати оцінку

// Відгуки
setBookReview(bookId, reviewText)  // Встановити відгук
getBookReview(bookId)              // Отримати текст відгука
getBookReviewDate(bookId)          // Отримати дату відгука
```

## ⚡ Горячие советы

## 🎨 Внешний вид

### Вкладка "📝 Заметки"
- ✍️ Большое текстовое поле для ввода заметок
- 📋 Список всех добавленных заметок в порядке добавления
- 🗑️ Быстрое удаление каждой заметки
- ✏️ Редактирование сразу в модальном окне

### Вкладка "⭐ Оцінка"
- ⭐ Интерактивные звезды для выбора оценки (hover эффекты)
- 💬 Большое текстовое поле для отзыва
- 💾 Кнопка сохранения отзыва
- 📅 Дата создания отзыва автоматически сохраняется

---

## 🐛 Известные ограничения

1. **Только после прочтения** - Оценку и отзыв можно дать только к прочитанным книгам
2. **Локальное хранилище** - Данные хранятся только в браузере (нет облачной синхронизации)
3. **Максимальный размер заметок** - Зависит от свободного места в localStorage (~5-10MB)
