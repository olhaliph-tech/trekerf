## 🔄 Жизненный цикл функций

### Добавление заметки:
```
1. Пользователь вводит текст в textarea
2. Нажимает кнопку "➕ Додати заметку"
3. addNote() создает новый объект
4. Сохраняется в book.notes[]
5. createActivityRecord() логирует действие
6. saveStateToStorage() сохраняет в localStorage
7. UI обновляется автоматически
```

### Установка оценки:
```
1. Пользователь нажимает на звезду
2. setBookRating() проверяет статус книги
3. Если статус = "прочитано" → сохраняет
4. Иначе → выводит ошибку
5. createActivityRecord() логирует
6. UI обновляется с новой оценкой
7. Все звезды обновляют свой стиль
```

### Сохранение отзыва:
```
1. Пользователь вводит текст отзыва
2. Нажимает "💾 Зберегти відгук"
3. setBookReview() сохраняет текст
4. Автоматически устанавливается reviewDate
5. createActivityRecord() логирует
6. saveStateToStorage() сохраняет
7. Сообщение об успехе: "✅ Отзыв сохранен!"
```

## 💾 Сохранение данных

### Автоматическое сохранение:
- Каждая операция вызывает `saveStateToStorage()`
- Данные сохраняются в `localStorage.readingTrackerData`
- Книги содержат notes, rating, review в JSON

### Загрузка при запуске:
```javascript
loadStateFromStorage()
  ↓
Парсит JSON из localStorage
  ↓
Инициализирует book.notes, book.rating, book.review
  ↓
Готово к использованию
```
