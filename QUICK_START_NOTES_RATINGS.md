# ⚡ Быстрый старт: Заметки и Оцінки (2 хвилини)

## 🎯 За 2 хвилини до першої заметки

### Крок 1: Відкрийте книгу ✅
```
1. Перейдіть на Бібліотека
2. Натиснувше на будь-яку книгу "Деталі"
```

### Крок 2: Перейдіть на вкладку "Заметки" ✅
```
Ви бачите 4 вкладки:
  📖 Бібліотека
  ⏳ Читання
  📝 Заметки ← СЮДИ!
  ⭐ Оцінка
```

### Крок 3: Додайте першу заметку ✅
```
1. Напишіть в текстовому полі, наприклад:
   "Цікавий фрагмент на сторінці 23"
   
2. Натиснувше на "➕ Додати заметку"

3. Вуаля! ✨ Заметка додана!
```

---

## ⭐ За 30 секунд до першої оцінки

### Як та коли оцінити?

1. **Статус повинен бути "Прочитано"** ← ВАЖЛИВО!
   - Змінити на вкладці "ℹ️ Інформація"

2. **Натиснути на вкладку "⭐ Оцінка"**

3. **Виберіть кількість зірок (1-5):**
   - ⭐⭐⭐⭐⭐ = 5/5 (відмінна)
   - ⭐⭐⭐⭐ = 4/5 (дуже добра)
   - ⭐⭐⭐ = 3/5 (добра)
   - ⭐⭐ = 2/5 (задовільна)
   - ⭐ = 1/5 (слаба)

4. **Оцінка встановлена!** ✅

---

## 💬 За 1 хвилину до першого відгука

### Напишіть своє враження

1. **На тій же вкладці "⭐ Оцінка"**

2. **Увидите поле "Ваш відгук"**
   - (видиме тільки для прочитаних книг)

3. **Напишіть що-небудь, наприклад:**
   ```
   Чудова книга! Головний герой дуже симпатичний,
   а сюжет тримає у напруженні до кінця.
   Рекомендую всім любителям детективів!
   ```

4. **Натиснувши "💾 Зберегти відгук"**

5. **Готово! 🎉**

## 📞 Що робити, якщо щось не працює?

1. **Заметка не зберігається?**
   - Перевірте, чи заповнене текстове поле
   - Натисніть кнопку "➕ Додати заметку"

2. **Можна поставити оцінку, але вона не зберігається?**
   - Перевірте статус книги (має бути "Прочитано")
   - Спробуйте ще раз

3. **Відгук не зберігається?**
   - Переконайтеся, що натиснули "💾 Зберегти відгук"
   - Не переходіть на іншу вкладку до збереження

4. **Все видалилось після перезарядки?**
   - Це можливо, якщо localStorage очищено
   - Спробуйте відкрити DevTools (F12) і перевірити Application → Local Storage
