# 📚 Швидкий старт - Трекер читања

## 🚀 Що нового?

### ⭐ **Рейтинги книг**
Оцініть прочитану книгу від 1 до 5 зірок:
1. Нажміть "Деталі" на книзі
2. Перейдіть на таб "⭐ Рейтинг"
3. Натисніть на зірки
4. Готово! Рейтинг збережеться

### ↕️ **Сортування**
Наприклад, переглянути всі книги за рейтингом:
1. На вкладці "Бібліотека" знайдіть "↕️ Сортування"
2. Виберіть "⭐ Рейтинг"
3. Список сортується автоматично

**Варіанти:**
- 📅 По даті (нові/старі)
- 🔤 По автору
- 📖 По назві
- ⭐ По рейтингу
- 📄 По сторінках

### 💬 **Рецензії**
Залишіть рецензію на книгу:
1. Відкрийте книгу → таб "⭐ Рейтинг"
2. Напишіть текст в полі "Оставить рецензію"
3. Натисніть "Отправить рецензію"
4. Переходьте на таб "💬 Рецензії" для перегляду

### 🎯 **Цілі читання**
Встановіть мету (наприклад, прочитати 10 книг у 2025):
1. Вкладка "Аналітика"
2. Прокрутіть вниз до "🎯 Цілі читання"
3. Введіть назву, кількість книг і сторінок
4. Натисніть "Добавить цель"
5. Смужка автоматично показує прогрес!

### 🏆 **Досягнення**
Отримуйте значки за досягнення:
- 📖 Першочитець (1 книга)
- 📚 Фанат (10 книг)
- 🏆 Легенда (50 книг)
- 📄 Тисячник (1000 сторінок)
- 🔥 Невгасна пристрасть (7 днів читання)
- ⭐ Цінитель (оцінка 4+)
- 🎨 Многогранник (5 жанрів)

**Як переглядати:**
1. Перейдіть на вкладку "👤 Профіль"
2. Прокрутіть вниз до "🏆 Ваші досягнення"

### 📚 **Статистика по жанрах**
Дивіться аналітику по жанрам:
1. Вкладка "Аналітика"
2. Знайдіть "📚 Статистика по жанрам"
3. Кожен жанр показує: всього / прочитаних / середня оцінка

### 🔥 **Популярні в спільноті**
Які книги найбільш активні:
1. Вкладка "👥 Спільнота"
2. На початку сторінки "🔥 Популярные в общине"
3. Топ-5 найбільш обговорюваних книг

### 👤 **Профіль користувача**
Оновіть свій профіль:
1. Перейдіть на "👤 Профіль"
2. Натисніть "✏️ Редагувати профіль"
3. Змініть:
   - Ім'я
   - Нікнейм
   - Біографію
   - Аватарку (посилання на зображення)
4. Натисніть "💾 Зберегти зміни"

## 💡 **Поради**

- 📌 Установіть цілі читання на початку року - це мотивує!
- 🌟 Оцінюйте книги - це допомагає іншим читачам
- 📖 Пишіть рецензії - поділіться своєю думкою
- 🔍 Використовуйте фільтри та сортування для кращої організації

## ❌ **Щось не працює?**

Придивіться:
1. Оновіть сторінку (F5)
2. Очистіть кеш браузера (Ctrl+Shift+Delete)
3. Перевірте, чи запущено сервер (localhost:3000)
